Premium, refined, technical-but-warm. Navy base (`#073B4C` / `#0B2A36`), red (`#D62828`) as restrained CTA/accent only. Neutral greys for surfaces and text. Linear brand icons. Real photography with navy overlay. Generous spacing, clear hierarchy, consistent grid.

## Motion Direction
Restrained: on-scroll reveal (fade + 16px rise), refined card hover elevation, animated counters on metrics, subtle CTA micro-interaction. Respect `prefers-reduced-motion`. No heavy libraries, no WebGL.
