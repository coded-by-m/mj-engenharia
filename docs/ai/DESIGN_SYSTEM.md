# Design System Foundations

> Generated by Coded by M — Site Architecture System. Source of truth: `/docs/ai/DISCOVERY.md`, `/docs/ai/DECISIONS_LOG.md` and the supplied brand manual.

## Visual Direction
Premium, refined, technically credible but human. Navy authority + clean neutral surfaces + restrained red signal. Engineering language expressed through a quiet technical grid, linear iconography, real photography with navy overlays, and disciplined spacing. Never loud, never alarmist.

Semantic mapping:
- Background base: `--white`; alternate section: `--mist-50`; inverse section: `--navy-700`/`--navy-900`.
- Text: primary `--graphite-800`, secondary `--grey-500`, on-dark `--white`.
- Brand/primary: `--navy-700`. Accent/CTA: `--red-600` (hover `--red-800`).
- Borders: `--grey-200`.

**Usage rule:** red is reserved for CTAs and tiny signal accents. Do not flood sections with red — it carries the fire-safety signal sparingly. Maintain AA contrast (navy/white is strong).

## Typography
Google Fonts via `next/font`:
- **Display / headings:** Montserrat (ExtraBold/Bold). H1–H3, hero, section titles.
- **Body / UI:** Inter (Regular/Medium/SemiBold). Paragraphs, cards, forms, nav.
- **Technical labels:** Barlow Condensed (Medium/SemiBold). Eyebrows, metric numbers, tags, data labels — uppercase, letter-spaced.

Type scale (fluid, `clamp`-based suggested):
- H1 ~clamp(2.25rem, 5vw, 3.75rem), Montserrat ExtraBold, tight leading.
- H2 ~clamp(1.75rem, 3.5vw, 2.5rem).
- H3 ~1.25–1.5rem.
- Body ~1rem–1.125rem, leading 1.6.
- Eyebrow/label ~0.75–0.875rem, Barlow Condensed, uppercase, tracking +0.08em.

## Spacing Scale
Base unit 4px. Scale: 4, 8, 12, 16, 24, 32, 48, 64, 96, 128. Section vertical padding: 64–128px desktop, 48–72px mobile. Generous whitespace is part of the premium feel.

## Containers and Grid
- Max content width ~1200–1280px, centered, gutters 24px (16px mobile).
- 12-column conceptual grid. Soluções: 3 cols desktop / 2 tablet / 1 mobile. Diferenciais: up to 5 across (or 3+2). Especialidades: 2 equal blocks.
- Consistent vertical rhythm; align everything to the grid — disciplined alignment is the main UX upgrade vs. the reference site.

## Radius and Shadows
- Radius: cards/inputs `--radius-md` 12px; buttons 8–10px; large media 16px. Keep sober.
- Shadows: subtle, layered. Resting card: soft 1px border (`--grey-200`) + faint shadow. Hover: slightly stronger shadow + 2–4px lift. Avoid heavy drop shadows.

## Essential Components
- **Navbar:** fixed, condenses on scroll; logo (horizontal positive), anchors, WhatsApp CTA (red). Mobile: hamburger + visible WhatsApp.
- **Footer:** navy background, mono/negative logo, anchors, contacts, social, copyright.
- **Buttons:** Primary = red fill, white text (WhatsApp CTAs). Secondary = navy outline/ghost. Tertiary = text link with underline-on-hover. Clear focus ring.
- **Links:** navy, underline on hover/focus.
- **Containers / Section headers:** eyebrow (Barlow Condensed) + H2 (Montserrat) + supporting line (Inter).
- **Cards:** icon (brand SVG) + title + one-liner; border + hover elevation. Especialidades cards larger with deliverable list.
- **Form fields:** labeled, comfortable height, visible focus, error states; dropdown for tipo de serviço.
- **Tags/badges:** Barlow Condensed uppercase (e.g., "CREA", "PPCI", "AVCB").
- **Floating WhatsApp button:** fixed, red, icon + label, accessible name.

## Responsive Principles
Mobile-first. Stacks to single column; CTAs become full-width; header collapses; touch targets ≥44px; no horizontal overflow; images responsive via `next/image`.

## Asset Guidelines
- Logos/icons: use provided SVGs (`Identidade Visual/Logo/SVG`, `/Icones/SVG`).
- Favicons/avatar: use provided sets.
- Photography: real project photos preferred; until available use placeholders with navy overlay; meaningful alt text.

## Reference Principles
Structure inspired by brandventuraengenharia.com.br, but elevated: stronger hierarchy, consistent grid/alignment, intentional whitespace, restrained color, specialist (PPCI) framing instead of generalist. Avoid generic template aesthetics.
