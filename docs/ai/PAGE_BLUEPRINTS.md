### Responsive behavior
- Desktop: multi-column grids (Soluções 3-col, Diferenciais up to 5/grid), generous whitespace.
- Tablet: 2-col grids; Especialidades blocks stack or stay side-by-side if space allows.
- Mobile: single column; header collapses to logo + WhatsApp + hamburger anchors; floating WhatsApp prominent; CTAs full-width; counters/animations lightweight.

## Main Conversion Page Blueprint
Same as Home (single page). Conversion redundancy: WhatsApp CTA in header, hero, intermediate band, final CTA, footer, and floating button. Form is the secondary capture path; on success show inline confirmation (or `/obrigado`).

## Services / Products Blueprint
"Soluções" + "Especialidades" together act as the services architecture:
- Two flagship deep blocks (PPCI, SPDA) with deliverable lists.
- A card grid for the full service set using the brand's 12 linear icons.
- Each card: icon + title + one-line value + optional anchor; hover = refined elevation.
- Card content uses real service names; deliverable specifics use placeholders where exact scope/wording is unconfirmed.
